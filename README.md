UI/UX Project 1 – Empathy Map

-Project Overview:
This project was completed as part of the **DecodeLabs UI/UX Industrial Training Program (Batch 2026). The objective was to conduct user research before beginning the interface design process by understanding the needs, goals, and pain points of the target users. Instead of designing screens, this project focuses on **empathy-driven UX research** through the creation of a User Persona and an Empathy Map.

-Project Objective

The goal was to analyze the target audience for a fitness application aimed at busy office professionals and identify their:
* Goals
* Motivations
* Frustrations
* Behaviors
* Thoughts and emotions

This research serves as the foundation for future UI/UX design decisions.

-Target Audience
Busy office professionals aged 25–40 who:
* Have demanding work schedules
* Find it difficult to maintain a consistent fitness routine
* Want quick and convenient workout solutions
* Use smartphones regularly for productivity and health-related tasks

-Deliverables
This project includes:
* Target Audience Analysis
* User Persona
* Empathy Map
  * 👀 See
  * 💭 Think
  * ❤️ Feel
  * 🏃 Do

-Tools Used:
* Canva (Presentation Design)
* UX Research Methods
* GitHub (Project Repository)

-Key Learning Outcomes
Through this project, I learned how to:
* Conduct basic UX research
* Build user personas
* Create empathy maps
* Identify user pain points
* Understand the importance of designing with users in mind before creating interfaces

-Future Improvements
In future stages, this research will be used to:
* Create user flows
* Develop wireframes
* Design high-fidelity UI screens
* Build an interactive prototype
