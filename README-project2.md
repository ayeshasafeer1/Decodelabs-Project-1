UI/UX Project 2 – Low-Fidelity Homepage Wireframe

-Project Overview
This project was completed as part of the **DecodeLabs UI/UX Industrial Training Program (Batch 2026)**.
The objective was to create a **low-fidelity wireframe** for an e-commerce homepage by focusing on user flow, information hierarchy, and usability. Rather than emphasizing colors or visual aesthetics, this project concentrates on the structural layout that guides users through the shopping experience.

-Project Objective
The scenario involved an e-commerce platform experiencing customer drop-off due to a cluttered and confusing interface.
The goal was to redesign the homepage structure by:
* Improving navigation
* Organizing content logically
* Creating clear Call-to-Action (CTA) buttons
* Establishing a simple and intuitive user journey

-Deliverables
This project includes:
* Homepage Low-Fidelity Wireframe
* Navigation Bar
* Hero Section with CTA
* Product Categories
* Featured Products Section
* Footer Layout

-Tools Used
* Figma
* Low-Fidelity Wireframing Techniques
* GitHub

-Design Principles Applied
* Information Hierarchy
* User-Centered Design
* Visual Consistency
* Simplicity
* Clear Navigation
* Call-to-Action Placement

-Wireframe Features
The homepage wireframe contains:
* Navigation menu
* Search bar
* Promotional banner
* "Shop Now" CTA button
* Product categories
* Featured product cards
* Footer with essential links

The design intentionally uses only grayscale elements to maintain a low-fidelity style and keep the focus on layout and functionality.

-Key Learning Outcomes
Through this project, I learned how to:
* Plan page layouts before visual design
* Create effective information hierarchies
* Design intuitive user flows
* Build low-fidelity wireframes in Figma
* Understand the importance of structure before applying visual styling
